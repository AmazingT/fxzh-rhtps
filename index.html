    Advertising design

    Advertising design assistant

    1、想获得一份有长远发展、稳定、有晋升空间的工作，有较强的学习意识和能力。

    2、19-28岁，大专及以上学历，喜欢从事设计类工作，对互联网行业感兴趣。

    3、可接受应届生和转行求职人员，只要有潜力可老带新，由项目经理带领岗前训练，但要求勤奋好学，积极主动。

    4、对平面设计/广告设计设计等有浓厚的兴趣。

    5、服从公司管理，认真学习和全面了解公司的相关情况和工作细节。

    薪酬福利

    1、工作时间：早9：00晚5：30，中午休息2小时，周末双休。

    3、公司文化：注重对员工能力和个性化培养，资助员工创新。

    4、发展空间：公司提供职业发展平台，公平合理的晋升发展机制。

  • 软件工程师

    JAVA大数据开发定岗生

    Software engineer

    JAVA big data development

    职位描述

    招聘：国企JAVA/大数据开发工程师及实习生20名(零基础无专业限制，有意想转行者请联系招聘经理）

    行业前景：随着大数据产业在全球范围内的快速兴起，国内互联网行业和IT行业的领军企业纷纷进军大数据行业。然而，由于人才培养的滞后，以及新兴行业熟练技术人员积累的不足，大批产业发展所需专业人才严重短缺。以Hadoop开发工程师为例，Hadoop入门薪资已经达到了 8K-15K ，具有2-3年工作经验的hadoop人才年薪可以达到 30万—50万。

    薪资待遇：银行、金融机构、国企、电信、上市科技公司等，从事大数据工作。行业前景非常可观，工作2年后工资在20000元以上。

    任职资格

    1、基于hadoop生态的大数据存储平台搭建和部署;

    2、完成日常数据分析查询需求；

    3、数据仓库的设计，开发，维护；

    4、参与大数据分布式应用系统服务器端或客户端软件开发工作（需求开发、故障解决和性能优化等）；
